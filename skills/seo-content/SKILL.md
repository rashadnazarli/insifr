# SEO Content Skill

When executed, you act as an SEO strategist and content writer producing search-optimised content grounded in Founder Mode artifacts.

## Execution Steps:
1. Read the following artifacts:
   - `docs/USER_PERSONAS.md` — search intent per persona
   - `docs/BRAND_GUIDELINES.md` — voice and tone
   - `docs/GROWTH_STRATEGY.md` — priority channels and keyword targets
   - `docs/PRODUCT_STRATEGY.md` — core value proposition
2. Identify the primary keyword cluster for the content request:
   - 1 head keyword (high volume, high competition)
   - 3–5 long-tail keywords (conversational, intent-driven)
3. Produce the requested content type:
   - **Landing page SEO copy:** Title tag, meta description, H1, H2 structure, keyword-rich body (800–1200 words)
   - **Blog post:** Keyword-optimised post (1200–2000 words) with internal linking suggestions
   - **Meta tags pack:** Title + description for all main pages
   - **FAQ schema content:** 5–8 Q&As targeting long-tail queries
4. Validate the output:
   - Keyword density 1–2% — no stuffing
   - Readability: Flesch score >60 (plain language)
   - Every claim backed by a product artifact (no invented features)

## Anti-Patterns:
- ❌ Keyword stuffing
- ❌ Thin content (<400 words per page)
- ❌ Generic titles like "Best [Product] in 2025"
- ❌ Missing meta descriptions

## Output:
Markdown document with clearly labelled sections, ready for implementation.
